Android Group Chat App

Android Platform

(Language: Java, PHP, MySQL)

Project Description

This project is a Chat application that allows a user to register a username and a password which is stored in a MySQL database. A PHP script is used by the Android app to store or retrieve data from MySQL. Once a user has registered, they can use their registered name and password to log in and chat with other users in an open public chat room.
