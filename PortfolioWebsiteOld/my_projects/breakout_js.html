Breakout - JavaScript

(Language and Technologies: HTML, JavaScript, Phaser Game Framework)

Project Description

This version of Breakout was created using HTML, JavaScript, and the Phaser HTML5 game framework. It is my first Game project in HTML and JavaScript and doing this project was a great way for me to learn and practice more JavaScript programming.

Try out the game here

Gameplay

The objective of the game is to break the bricks and gain points to get the high score. The game is over when you have zero lives left.
